== Key Features and Benefits
* [*] Superior AI Performance for production:
on-device processing with up to 100 TOPS AI
performance with low power and low latency

* [*] Edge AI device: includes NVIDIA Jetson Orin(TM) NX 16GB production module,
a cooling fan with a heatsink, enclosure,
and a power adapter. Supports DIN rail mounting.

* [*] Accelerate solutions to market:
pre-installed Jetpack with NVIDIA JetPack(TM)
5.1.1 on the included 128GB NVMe SSD,
Linux OS BSP, 128GB SSD,
supports Jetson software and leading
AI frameworks and software platforms

* [*] Wide temperature range - operating from 14°F to 140°F (-10°C to 60°C)
