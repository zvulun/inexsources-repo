[#s_Checking-Mounting-Screws]

=== Checking Mounting Screws

It is recommended to check all mounting screws for proper tightness once every two years.

[#s_Cleaning-the-Camera-System]

=== Cleaning the Camera System

Do not use solvents or strong abrasive detergent when cleaning the Camera System. Use a soft dry cloth to clean the Camera System's front glass when it is dirty. If the dirt has hardened, remove it using mild soap and water, and then wipe the front window +++<u>+++gently+++</u>+++.
