This document describes how to install,
configure and calibrate an {compname-short} {generic-camera-name} for a single lane, to work with the TIBA Parking System.

== Use {software-discovery} and Allocate IP Addresses



. Allocate 3 consecutive IP addresses for
the onboard Jetson processor and the
LPR and OV sensors.
For example: +
* Processor - http://192.168.5.100/ +
* LPR Sensor - http://192.168.5.101/ +
* OV Sensor - http://192.168.5.102/ +
